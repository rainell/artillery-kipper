# Klipper configuration files for Artillery printers
* The use of these files is recommended for advanced users.
* It is advised to verify these settings by following the official [Klipper documentation](https://www.klipper3d.org/Config_Reference.html).
* The user is solely responsible for anything that may happen using these settings.
* Do not forget to calibrate the extruder PID and the heater bed PID before starting to print.
